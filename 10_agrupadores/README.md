# Capitulo 10: Agrupadores

## Crear un agrupador en bloque de elementos

1. Agregar:

```
    <div>
      <h2>Titulo Secundario</h2>
      <p>Este es el contenido de este articulo</p>
      <img
        src="./assets/img/imagen.png"
        alt="panda recaliente en la oficina"
        height="300px"
      />
      <br />
      <a href="https://www.google.com" target="_blank">Ir a google</a>
    </div>
```

![Agrupador en bloque](agrupador_en_bloque.png)

## Crear un agrupador en linea de elementos

1. Agregar:

```
    <br />
    <span>
      <strong>La fecha del dia de hoy</strong>
      <em>3/7/2024</em>
    </span>
```
